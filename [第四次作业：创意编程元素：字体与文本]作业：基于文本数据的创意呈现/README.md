# 第四次作业
## task 1
### 按住鼠标并移动产生文字
- 鼠标移动速度越快，字越大
- 为避免连续的两个字相互重叠，设置了鼠标移动距离与字体大小的判断，只有移动距离足够大时才会显示下一个字
