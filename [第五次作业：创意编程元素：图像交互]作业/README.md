## 第五次作业
### 递归函数画笔
- 通过递归函数在鼠标按下时画出互相嵌套的圆
- 按1画笔为红色， 2为绿色，3为紫色，按其他键为灰色，默认为灰色
- 's'键保存并输出，'r'键清空
